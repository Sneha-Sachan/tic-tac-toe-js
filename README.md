# Tic Tac Toe Game

A simple 2-player Tic Tac Toe game built using HTML, CSS, and JavaScript.

## Features
- Playable by two players
- Win detection logic
- Restart button
- Minimal UI

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
Open `index.html` in any browser and start playing!

## Future Improvements
- Add AI mode (play vs computer)
- Scoreboard
- Animation effects
